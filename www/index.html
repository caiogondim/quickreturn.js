Installing

Use the minified version on your site. Don't worry about debugability, it has source maps. 

<script src="quickreturn.min.js"></script>

Create a new QuickReturn object, passing the element you want to behave with the QuickReturn pattern as the argument. Then, call init(). 

var navbar = document.querySelector(".navbar")
quickreturn = new QuickReturn(navbar)
quickreturn.init()

Whenever you want to come back to the default behavior, just call destroy() on the instance. 

quickreturn.destroy()
